# BlazorApplication1.git

## Prerequisites

* **.NET Core 2.0 preview 3** or later. I've been using [SDK build 6764](https://dotnetcli.azureedge.net/dotnet/Sdk/2.0.0-preview3-006764/dotnet-sdk-2.0.0-preview3-006764-win-x64.exe).

## Run

```
dotnet restore
dotnet blazor serve
```

http://localhost:5000/
